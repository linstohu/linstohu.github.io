# 项目说明

这是一个用 Hugo 构建的个人网站。核心内容是位于 `content/essays/` 下的中文随笔专栏。

## 文件与结构约定

- 每篇文章一个目录：`content/essays/<english-kebab-slug>/index.md`，slug 用英文短横线命名（如 `description-is-not-the-thing`）。
- Front matter 格式：
  ```yaml
  ---
  title: "中文标题"
  date: 2026-06-14T22:30:00+08:00
  lastmod: 2026-06-14T22:30:00+08:00
  ---
  ```
  `date`／`lastmod` 必须用**真实的当前日期时间**，时区固定 `+08:00`。不要凭空编一个时间（如随手写个 21:30、23:50），写之前先跑一条命令取真实时刻：`date "+%Y-%m-%dT%H:%M:%S+08:00"`，把输出原样填进去。新建文章时 `lastmod` 与 `date` 相同。
- 开头第一段（钩子段）之后，单独一行放 `<!--more-->` 作为摘要分隔符。
- **文章结构顺序**：front matter → 钩子段 → `<!--more-->` → 「写作说明」admonition → 「提示词」fold → 正文（`##` 小标题分段）。即「写作说明」和「提示词」都紧跟在 `<!--more-->` 之后、正文之前，而**不是**放在文末。
- 可用的 Hugo shortcodes：`admonition`（类型有 `note` / `info` / `warning` / `abstract` / `example`）、`center`、`left`、`right`、`fold`、`table`、`spotify`。引用提示框用 `{{% admonition note "标题" %}}...{{% /admonition %}}`。

## 写作风格（务必遵循）

- **第一人称、口语化、有态度**。像跟朋友掏心窝子讲话，可以带点情绪、自嘲和狠话（如"傻子都知道""屁话""中看不中用的王八蛋""立马尿"），但不油滑。
- **从一段真实的个人经历或当下感受切入**作为钩子，再引出论点。
- **大量使用类比**来讲道理（已用过的：司马光砸缸、大禹治水、拆钟表、游泳、地图与山、操作系统、树的年轮）。一个好类比胜过千言万语。
- **多用反问句**推进论证。
- 常**引用名家原话、书名或学者观点**（马克思、查理芒格、冯唐《孙子兵法》、Krashen 等），英文原句可用 blockquote，再给一句精炼的中文翻译。
- 善用 `##` 小标题分段；善用 `A → B → C` 这类箭头流程来对比"错误顺序 vs 正确顺序"，但不滥用。
- **结尾用一句点睛的短句收束**，留有余味。
- 篇幅：中等偏短，观点集中，不注水。
- 关键词、概念加书名号「」或 `**加粗**` 来强调。

### 容易写跑偏的地方（重点，写之前先过一遍）

- **钩子必须是带血带肉的具体真事，不是泛泛的感受**。要有具体的人（叔叔、邻居、父亲、初中同学）、具体的数字（24 万买、50 万的房、一千多万打水漂、两套空调换 iPhone）、具体的地点（衡水二中、河北、县城）。开头第一段就要把读者拽进一个画面里，而不是"我最近在想一个问题"这种悬空开场。
- **叙事要长段铺陈，别急着列点**。先把那件事原原本本讲透、讲出情绪，再从里头抽出道理。bullet 列表只在排比式地数落几条并列毛病时偶尔用（如"道德标准高 / 形式大于内容 / 责人严责己宽"），不要把整篇拆成整齐的小条目，那样就成了讲道理的范文，丢了劲儿。
- **自嘲和狠话要落到实处**，是"数数自己手里能拿来赚钱的武器吧，自己都乐了""真让他下场干一件，立马尿"这种带画面、带数字的具体自嘲，不是抽象地说"我也曾如此"。骂人骂自己都敢往狠了骂，但骂完要有反思的落点。
- **善用阶级/底层视角和"血淋淋的案例"**做论据：农民家庭、老实人吃亏、脑死亡、求职被坑。道理是从这些扎心的现实里逼出来的，不是从概念演绎出来的。
- **可以有"一问 / 二问 / 三问"层层追问的结构**，自问自答往深里挖。
- **可以引用、链接自己以前的文章**，用 `{{< relref "essays/<slug>" >}}`。
- **标题偏向一句直给的断言或狠话**，可带括号补一层身份/自指（如「痛恨一知半解的人（当年的自己）」「越靠近真相，越靠近牛逼（写给二十岁的自己）」），不要起成中性的说明文标题。
- 句子节奏可长可短、敢用口语虚词和语气（"啊""呗""罢了""人生啊…"），允许略带粗粝，不要为了通顺把棱角磨平。

## AI 写作署名与提示词留档

在 `<!--more-->` 之后、正文之前，紧挨着放两块内容：

1. **「写作说明」admonition**：讲清"想法素材是我的、成文润色由模型完成"这层分工。措辞每次可微调，不要机械照抄：
   ```markdown
   {{% admonition note "写作说明" %}}
   本文的想法、素材与观点均出自我本人，成文则由 Anthropic 的 Claude Opus 4.8 模型按我的语言风格整理润色而成。
   {{% /admonition %}}
   ```
2. **「提示词」fold**：把我交给模型的原始提示词/素材原样收录进一个可折叠块，方便日后回看成文与原始想法的差距。标题固定为「提示词（我交给模型的原始素材）」。内容用 `{{% fold %}}`（百分号形式，让里面按 markdown 渲染）原样照搬我的提示词，不要改写、精简或润色：
   ```markdown
   {{% fold title="提示词（我交给模型的原始素材）" %}}
   <这里原样粘贴我发给你的提示词全文>
   {{% /fold %}}
   ```
