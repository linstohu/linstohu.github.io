最近我开始学音乐，想着既然要学，那就系统点，先把乐理啃下来。结果一翻开书，我就觉得不对劲：一上来就让我背 C D E F G 和 do re mi fa sol la si 的对应关系。背着背着，那种熟悉的、令人作呕的感觉又回来了——这不就是高中六点起床背语文课文吗？背了忘，忘了背，兴趣被消磨得一干二净。

<!--more-->

我停下来，问自己：哪里出了问题？

后来我想明白了。语言学领域有位极有影响力的学者，叫 Stephen Krashen，他提出过一个核心观点：

> Language acquisition is different from language learning.

翻译过来就五个字：**习得 ≠ 学习**。

这句话像一根针，一下子扎破了我憋了很多年的一个疑惑。

## 我们学的，常常不是事物本身

我们嘴上天天挂着「学习」，可仔细想想，我们学的到底是什么？

很多时候，我们学的不是那个事物本身，而是**前人对那个事物的「描述」**。

你想啊，一门学问，几千年里多少聪明人扎进去，他们摸到了门道，悟出了窍门，然后呢？他们急切地想把这点心得传给后辈，于是绞尽脑汁发明各种术语、各种符号、各种规则，去描述他心里那点说不清道不明的东西。一代又一代，大家纷纷添砖加瓦，可不就越来越深奥、越来越“难啃”呗。

到最后，初学者一进门，迎面撞上的不是事物本身，而是这座由「描述」堆起来的高墙。

乐理就是这样。CDEFG 也好，五线谱也好，那都是音乐的「描述系统」，是前辈们为了把脑子里的声音记下来、传下去而发明的工具。它不是音乐本身。音乐本身是什么？是好听，是你忍不住跟着哼，是那段旋律在你脑子里挥之不去。可我一个连旋律都没在耳朵里待熟的人，张口就让我背符号对应关系，这不是本末倒置吗？

## 同一件事，在英语上演过一遍

这事儿我太熟了，因为它在我学英语的时候，已经原原本本演过一遍。

我们大多数中国学生学英语的路子是这样的：

> 语法 → 单词 → 语法 → 语法 → 做题

而不是这样的：

> 大量输入 → 理解 → 模仿 → 表达

于是就有了那个怪现象：一个学生能把 `If I had known...` 这种虚拟语气的题做得滴水不漏，却在外国人面前一句话都不敢开口。

这俩事儿，本质上是同一件事：**专家在教你「描述系统」，而你连「直觉系统」都还没建立起来。**

语法是什么？是语言学家对语言现象的事后总结，是「描述」。可一个孩子学说话，哪个是先学主谓宾再开口的？都是先听，听了几千遍上万遍，听熟了，自己就咿咿呀呀地模仿，然后慢慢就会说了。语法是他长大以后才回过头来知道的事，甚至一辈子不知道也照样说得溜。

## 顺序错了，再用功也是白费

我后来发现，这是个特别普遍的教育毛病。绝大多数学科的教学顺序是：

> 理论 → 理论 → 理论 → 实践

可人脑真正吃得消的顺序，恰恰反过来：

> 体验 → 熟悉 → 形成直觉 → 理论总结

就拿游泳打比方。

错误的教法是：先给你讲流体力学，再分析划水角度，再讲呼吸原理，讲完了，下水。

正确的教法是：先把你扔下水，先让你扑腾着漂起来，先让你能扑通扑通游两下，等你身体有了感觉，再回过头去优化动作。

哪个学得会？傻子都知道。可一到正经学知识，我们就鬼使神差地选了前一条路。

## 别在「描述」里沉醉

所以我想说的其实就一句话：**「描述语言」和「事物本身」之间，天生隔着一段距离。**

学东西的时候，如果你一头扎进「描述」里出不来，沉醉于那些术语、符号、规则，把它们背得滚瓜烂熟，你会有一种自己很用功、很扎实的错觉。可你离事物本身，其实越来越远了。

你确实在「学习」，这没错。但你学的不是那个事物，而是飘在事物上头的那层「描述」。

就像捧着一张地图研究了半天，把每条等高线都背下来了，却始终没有迈出家门一步。

地图画得再精细，也不是那座山。想认识那座山，先上去走一趟。
